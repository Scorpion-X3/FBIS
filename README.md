# FBIS
Fencing Bout Information System – a Python tool to log and analyze fencing bout data

## Features
- Validates bout type and score
- Tracks touches (attack, parry, remise, counter)
- Identifies most common touch
- Stores bout data in JSON format

## How to Run
1. Run main.py
2. Enter bout type (DE or Pool)
3. Enter score (e.g. 15-10)
4. Enter touches
5. Enter who scored each touch
